# HW5 — YouTube AI Chat Assistant — Dev Notes

## Repo summary (Phase 0)

- **Frontend**: React (react-scripts), recharts.
- **Backend**: Express (`server/index.js`), port 3001.
- **DB**: MongoDB (`chatapp`). Collections: `users` (username, password, email, first_name, last_name), `sessions` (username, messages[], channelData, uploadedImages).
- **System prompt**: `public/prompt_chat.txt` (loaded by `src/services/gemini.js`).
- **Chat**: Messages in `sessions.messages`; load/save via `src/services/mongoApi.js`. AI: Gemini via `src/services/gemini.js` (streamChat or tool-calling). CSV tools run client-side in `src/services/csvTools.js`; YouTube tools run server-side via `/api/tools/*`.

## Key files changed

- `server/index.js` — first_name/last_name in users; login returns them; sessions store channelData/uploadedImages; `/api/channel/download`, `/api/upload/json`, `/api/upload/image`, `/api/tools/*` (generateImage, plot_metric_vs_time, play_video, compute_stats_json).
- `src/components/Auth.js` — first_name, last_name inputs on Create Account.
- `src/services/mongoApi.js` — createUser/login payloads; getSession; uploadJson, uploadImage, getChannelData.
- `src/App.js` — user object { username, first_name, last_name }; tab: Chat | YouTube Channel Download.
- `src/components/Chat.js` — user name in context; JSON drop zone; channelData state; YouTube tool-calling path; render image/plot/videoCard/stats.
- `src/services/gemini.js` — load prompt; YouTube tool declarations; chatWithYouTubeTools (server-side tool execution via fetch).
- `public/prompt_chat.txt` — YouTube assistant persona; TOOLS AVAILABLE (generateImage, plot_metric_vs_time, play_video, compute_stats_json); first-message greeting instruction.
- `public/veritasium_channel_data_10.json` — sample channel JSON (generated by YouTube download).

## How to run

1. `cp .env.example .env` and set `REACT_APP_GEMINI_API_KEY`, `REACT_APP_MONGODB_URI`.
2. `npm install && npm start` (server on 3001, client on 3000 via proxy).

## New env vars

- Optional: `OPENAI_API_KEY` or `REACT_APP_OPENAI_API_KEY` for generateImage (DALL-E 3). If unset, anchor image is returned when provided, else error.

## Acceptance checklist (HW5)

1) Sign up stores first/last name and AI greets by name on first message.
2) "YouTube Channel Download" tab works and shows progress.
3) It produces downloadable JSON.
4) public/veritasium_channel_data_10.json exists and can be downloaded from the running app (or generated by running download with Veritasium URL, 10 videos).
5) Drag/drop JSON into chat loads it and AI acknowledges it.
6) generateImage works with prompt + optional anchor image and renders enlarge + download.
7) plot_metric_vs_time renders chart in chat with enlarge + download.
8) play_video renders clickable video card and opens new tab.
9) compute_stats_json returns correct stats and renders stats block.
10) public/prompt_chat.txt documents ALL tools and YouTube assistant behavior.
